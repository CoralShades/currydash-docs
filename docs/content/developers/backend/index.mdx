# Backend Development

Technical documentation for the CurryDash Laravel backend.

## Technology Stack

> **Note**: Verify versions against production `composer.json`. Last verified: 2026-01-22

- **Framework**: Laravel 9.x+ (PHP 8.1+)
- **Database**: MySQL 8.0+ (Cloud SQL)
- **Cache**: Redis (Memorystore)
- **API**: RESTful JSON API

## Quick Start

```bash
git clone https://github.com/CoralShades/CurryDash-Admin-Seller-Portal.git
cd CurryDash-Admin-Seller-Portal/backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
