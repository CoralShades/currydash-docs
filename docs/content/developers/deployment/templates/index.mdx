# Deployment Templates

Pre-configured deployment templates for deploying various application types to Google Cloud Platform.

## Available Templates

- **[Flutter Web](/developers/deployment/templates/flutter-web)** - Deploy Flutter web applications to Firebase Hosting or Cloud Run
- **[Laravel](/developers/deployment/templates/laravel)** - Deploy Laravel applications to Cloud Run with Cloud SQL
- **[Next.js](/developers/deployment/templates/nextjs)** - Deploy Next.js applications to Cloud Run or Firebase
- **[Node.js](/developers/deployment/templates/nodejs)** - Deploy Node.js applications to Cloud Run
- **[React](/developers/deployment/templates/react)** - Deploy React SPA applications to Cloud Run or Firebase Hosting

## What's Included

Each template provides:
- Deployment architecture and options
- Cost estimates for different configurations
- Quick start commands
- Docker configurations
- CI/CD pipeline examples
- Environment configuration guides

## Usage

1. Navigate to the template that matches your tech stack
2. Review the deployment options and choose one that fits your needs
3. Follow the quick start guide to deploy
4. Customize the configuration files for your project
