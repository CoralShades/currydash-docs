## Admin Portal Access

Access the Admin Portal at: **https://admin.currydash.com**

## Key Functions

- **User Management**: Create, edit, suspend user accounts
- **Vendor Onboarding**: Review and approve vendor applications
- **Order Oversight**: Monitor orders, handle disputes
- **Financial Reports**: Revenue, commissions, payouts
- **Platform Settings**: Zones, fees, promotions
