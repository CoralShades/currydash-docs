# Guides

Step-by-step guides for common tasks and workflows.

## Available Guides

Guides are automatically generated from completed stories. Check back as more features are documented.

## Contributing

Documentation is generated from BMAD stories. When a story is completed:

1. Run `*paige da` to analyze documentation needs
2. Run `*paige ds` to sync documentation changes
3. New guides will appear here automatically
